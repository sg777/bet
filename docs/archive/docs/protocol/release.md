# Pre-compiled Binaries (legacy)

> **Deprecated.** This document pointed to `chips-blockchain/bet`,
> `chips-blockchain/chips`, and `chips-blockchain/lightning` release
> archives. None of those archives are produced any more — the
> Lightning Network dependency and the CHIPS blockchain dependency
> have both been removed from `bet`.
>
> There is **no** official pre-compiled binary release for the
> Verus-era `bet` at this time. To run a node, build from source:
>
> * `bet` itself: see [`compile.md`](../../../guides/build-from-source.md).
> * The Verus daemon and CLI: build from the upstream
>   [`VerusCoin/VerusCoin`](https://github.com/VerusCoin/VerusCoin)
>   repository, or apply the `-offline` patch described in
>   [`docs/VRSCTEST_LOCAL_SETUP.md`](../../../tutorials/cli-auto-vrsctest.md) for
>   the regtest variant.
>
> This file is kept to preserve inbound links during the
> documentation reorganisation and will be archived under
> `docs/archive/` in a follow-up cleanup.
